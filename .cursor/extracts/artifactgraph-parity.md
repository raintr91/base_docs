# ArtifactGraph — parity and context-orphan

- `parity-drift`: create/edit/API labels, validation, type, or empty policies
  disagree. Present A/B/C and remember the member-confirmed canon.
- `context-orphan`: an action uses data outside the screen's displayed data
  scope. Warn only; do not turn it into a confirmation gate.
- Run `artifactgraph_parity_check` locally and send only unresolved structured
  findings in a cloud slice.
